## Development Setup

1. **Fork & Clone**
   ```bash
   git clone https://github.com/YOUR-USERNAME/ASA-SERVER-MANAGER-2.0.git
   cd ASA-SERVER-MANAGER-2.0
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run tauri dev
   ```

## Code Style

- **TypeScript**: Use strict TypeScript with proper types
- **React**: Functional components with hooks
- **Rust**: Follow Rust conventions, run `cargo fmt` before committing
- **Commits**: Use conventional commits (feat:, fix:, docs:, etc.)

## Pull Request Process

1. Create a feature branch from `main`
2. Make your changes with clear commits
3. Test thoroughly on Windows
4. Update documentation if needed
5. Submit a PR with a clear description

## Reporting Issues

- Use the issue templates when available
- Include your OS version and app version
- Provide steps to reproduce the issue
- Attach logs if relevant

## Questions?

Open a discussion or reach out via issues!
